## 摘录
```
为了试用E-M5III我购入了12-100f4这支镜头，为了配合这支镜头和更新手中老旧的GX7，又购入了去年出的GX9。这大半个月来连续使用这几台机器，有些使用感受的对比很想和大家一起分享一下。但是重新码一篇文章又太费时间，于是我就偷懒在我的E-M5III评测基础上添加GX9相关感受甚至一些些和GX7相关的内容。为了内容连续性，请允许我用长篇幅的纯文字来完成这次分享，至于E-M5III的图片，可以参考我在无忌的试用贴。而GX9的照片会后续慢慢贴出来。

  对我而言，M43是一个曾经非常熟悉近日却又有些陌生的系统。从10年开始接触无反，至今至少有6年是靠着M43系统的陪伴而度过的。只是就最近的三年而言，M43相机用得却相对非常少。曾经手里一众M43镜头（主要是一些小巧的定焦头）也仅剩下了一支15mm F1.7还在我的防潮柜里。所以在这个系统里我是渐渐落伍的，直到今天拿到这台奥林巴斯的新作E-M5 Mark III。

  曾经的M43是小巧快速的代名词。这么多年来各大厂商在其他无反系统上不断发力，如今的这台装备了奥林巴斯旗舰技术的E-M5 Mark III是否依旧可以在对焦能力上傲视群雄？之前之所以离开M43，主要是因为画质上的差距，尤其是高感的表现。时隔多年，M43的底是否能带来可喜的画质变化？也许这是这次拿到这台还未上市（拿到相机的时候）的奥林巴斯新作时我最想去了解的两个部分。如果考虑到奥林巴斯E-M5 Mark III所使用的技术和传感器大多数还是来自于三年前的旗舰机的话，那么松下的GX9作为一年前的新机应该也不算太老旧，而且这也是2000万像素的传感器，虽然不知道这块传感器和奥林巴斯那块相近像素的传感器有何渊源，但至少目前松下的旗舰用的也是这块传感器。那么画质上是否也是相近的呢？而且两者有着相似的身材，对我这种经常扫街的人来说是个比较合适的大小。

  感谢色影无忌编辑部，在给我相机的时候不仅很贴心地给了我两块电池，也随机提供了45mm F1.2 PRO 这款高素质的PRO系列定焦。考虑到试用期间正好要飞跃重洋去墨西哥的瓜达拉哈拉出差，于是我又立马购进了一支 12-100mm F4 IS PRO 高端恒定光圈大变焦“旅游”镜头（我愿意用旅游镜头来描述这支神奇的12-100mm F4 IS PRO，但在奥林巴斯的镜头序列里，它可是一支不折不扣代表高端品质的PRO系列镜头）。加上手头的这只15mm F1.7，也大致算是凑够了这次试用的镜头群。其实，用回M43我的内心还是有些忐忑的。这些年用惯了大底的机器和一众大光圈镜头之后，是否还能用M43那略许欠缺的虚化能力来很好地表达画面的主体？更何况是我并不太熟悉的奥林巴斯系统。

的确，这台机器的确花了我几天的时间去上手和习惯，更不用说那个给工程师用的超级详细和复杂的用户菜单了。好在我还勉强算是一个工程师，好在我也里里外外接触过近百台各色相机了。一周下来，大多数设置已经被我安排在了相对比较适合我个人使用习惯的设定上了。接下来就该出门正儿八经地拍照了。

E-M5 Mark III 装备了从旗舰E-M1 Mark II 下放的绝大多数功能，同时也因为PEN系列的存在，所以在体积上这台E-M5 Mark III 并没有体现出太明显的M43系统的小巧优势，甚至于还略略大过于上一代产品。但也正因为如此，机身在握持手感或者操控按钮的便利性上用户基本上没有什么可以挑剔的。加上内容丰富的触摸屏，基本上我日常使用所需要调整的功能都可以非常方便地通过按钮或者波轮来快速设置。而且各个按键的位置安排合理，很少会导致误触摸。如果非要说有什么不太合理的地方，那么录音外设和遥控插孔的位置和翻转屏的干涉可能算是为数不多的在使用中遇到过的问题。E-M5 Mark III 的翻转屏是外翻的，所以在低角度使用遥控器拍摄的时候，插入的遥控头子会导致翻转屏无法方便翻转，只能先翻转好屏幕再插入遥控器。此外我朋友在使用这台相机的时候曾抱怨过相机开关位置在左侧的设计，我想这台相机从体积来看就是设计给双手使用的，所以它并没有兼顾单手使用的需求。其实菜单里也可以将右侧取景器旁的拨钮设置成为开关的选项，只是那个拨钮拨杆有点短，操作起来并不是太轻巧。额外点赞的是奥林巴斯的EVF和LCD显示切换，距离感应器工作逻辑本身就非常聪明，而且除了有切换按钮可以直接操作，还能在菜单内根据屏幕开合的状况进行选择，能够完全避免因距离感应器捣乱而影响拍摄的问题。相对于奥林巴斯，我对于松下的M43相机却是很熟悉的，所以上手并没有什么难度。这次GX9终于将无反相机最常用的曝光补偿拨盘作为一个独立的拨盘放在了机身右侧顶部，相对于之前需要将机身背部波轮定义为曝光补偿相比，要更直观省事很多。结合DG15的镜身光圈环，使得我们可以在不开机的情况下预调整好我们需要的拍摄参数，可以在开机后快速进入拍摄的状态。这点和我这两年最喜欢用的富士系统有一定的相似性。此外在Ai模式下，这个曝光补偿一样可以起作用，非常实用。当然这个拨盘在使用的时候还需要习惯，因为机身背部波轮还是存在，在盲操作的时候会形成一定的干扰。也因为这个拨盘的存在，电源开关少许远离了右手拇指，操作略有不便。GX9终于在自动ISO的设置上向主流靠拢了，这也使得扫街的设置变得非常简单，这是我在这台机器上看到的最开心的事情。但是GX9的EVF着实太落后了，分辨率不高，画面小，偏色等问题往往会导致对最终画面判断的偏差。这点E-M5 Mark III无疑要做得好许多。

不知道是不是因为机身五轴防抖比较耗电还是其他原因，E-M5 Mark III 的电池似乎并不是很耐用。我在墨西哥闲逛的一天里，两节电池并没有让我撑过一天，基本上每节电池只能拍摄400张左右。要知道我的拍摄方式向来很省电，从不长时间取景，绝大多数情况下开机后的五秒内会完成拍摄。E-M5 Mark III 增加了USB充电功能，但是需要在关机的情况下才能使用，而且充电速度并不快。我在两节电池耗尽之后尝试了一下这个功能，在充电20分钟后显示有两格电，但是仅仅在拍摄了40多张照片后又再次显示电池耗尽。所以带E-M5 Mark III 外出有比较大的拍摄量的时候最好还是多带几节电池在身边。在耗电上GX9和E-M5 Mark III谁也不比谁好到那里去。GX9要比GX7更耗电，同样一节电池，GX9要比GX7少拍三分之一到四分之一的样子，和E-M5 Mark III可谓不分伯仲。而且GX9用USB充电也不够快，但是它能够在充电的时候打开相机拍摄（虽然此时充电会暂停，拍完关机后会自动继续充电）。相比E-M5 MarkIII只能在拔下充电线后才能打开相机而言要方便不少。相当时我在瓜达拉哈拉的街头不断插拔USB充电线，那才叫一个狼狈。顺便提一下，GX9的USB口的滑动式盖子质感很好，比橡皮塞子上档次不少。

奥林巴斯的五轴防抖一如既往地出色，虽然现在多家机身都加入了防抖功能，而且也有不少都称作为五轴防抖，不过迄今为止奥林巴斯的防抖还是独步天下的。单单依靠机身防抖，使用松下的 15mm F1.7就能做到1s曝光画面放大也没有明显的模糊，而此时画面中开过的汽车已经可以拉出一条占据大半个画面的尾灯红线了。而使用12-100mm F4 IS PRO 这支防抖镜头时，机身防抖和镜头防抖还能协同工作，据说可以达到6.5级的防抖效果。在实拍中即便在100mm端，1/2s手持曝光的拍摄结果也是令人满意的。所以当我晚上在瓜达拉哈拉老城区里转悠的时候，把光圈放在F4，ISO开到800，手持拍摄街景和建筑的时候，即便曝光时间经常在1/4~1/20s之间，画面还是相当靠谱的。（M43系统即便光圈在F4，使用广角拍摄的时候景深还是有保障的，加上ISO800对于这套系统来说画质也远没有开始明显劣化，所以对于要求不是太高的旅游夜景来说，完全不需要三脚架）我经常喜欢拍点带动态的片子，常常会把曝光时间放在1/4s~1/15s来让路上的行人在画面中动起来。用其他系统拍摄的时候常常会因为手持的不稳定而带来画面中固定的部分产生模糊，而用这套系统则完全不用担心，可以全心全意去关注画面中元素的位置并选择在最合适的时机按下快门。GX9虽然也号称五轴防抖，但是差距还是有点大，使用下来感觉有两档的差距，同样是DG15，奥林巴斯E-M5 MarkIII能在1秒达到的效果，GX9上需要1/4秒。这对于弱光下对于静态物品的拍摄就有了不小的差距。所以从旅游使用角度来说，GX9不如EM5适用性更广（而且EM5III三防）

奥林巴斯的色彩也一直是它的特点之一，尤其是在蓝色和黄色的表达上非常纯粹。如果说黑白是画面明暗层次的关键的话，那么冷暖中的蓝和黄则带来了丰富的色彩层次，在某种程度上也能使画面更加立体和生动。刚拿到相机那会儿，还不知道怎么解这台相机的RAW格式图像，一边熟悉相机一边用JPG拍了不少照片。在这个秋天里，奥林巴斯对秋叶色彩的表达令人印象深刻，从绿到红，既有些夸张的性格却又不致于艳俗，还带有些许厚重的感觉。当然E-M5 Mark III 也提供了多种色彩预设和风格的选择，只是更多依靠自己直觉后期的我并没有更多地去尝试。但相信对于喜欢直出的朋友来说，奥巴的色彩多半是不会让你失望的。虽然说这些年松下和徕卡走得很近，而且很多摄友说松下的色彩越来越有德味了，但我个人是不认同的。在某些颜色上，比如红色和绿色，松下的发色总让人觉得不太正，后期要调舒服不如奥林巴斯来得容易。当然，色彩这个东西还是比较主观，萝卜青菜各有所爱，就不多说了。
说完色彩，让我们再来看看M43用户最不愿意提及的画质吧。坦率地说，M43还是M43，来到两千万像素也未必会给你带来堪称特别细腻的画质。低ISO下直出和以往的M43不会有太大的区别，而在RAW下宽容度相比以往的机型有小幅提升，拉亮3EV色彩和细节的保真度还不错，也并没有带来太明显的噪点。高ISO下比我预想的要好不少，虽然有些细节损失，但ISO3200缩图至1600像素边长画面还是相当干净的。即便ISO6400会带来一些噪点，但对于出图的影响也不大，基本可用。当然高ISO下的宽容度就不能奢求了，ISO6400下的后期空间很小，基本上还是要拍摄者在前期做到基本的准确。ISO6400基本可用给这台机器带来了很大的实用性，因为即便是全画幅相机可用的ISO也大都在10000以内，但M43系统在相同体积下可以带来更大光圈的镜头，这让夜拍变得现实。这次拿到的 45mm F1.2 PRO 虽然体积上也不小，但是依靠1.2的光圈结合ISO6400，让我在小区游乐场这种夜间相当昏暗的场所也能以比较高的快门速度来抓拍孩子的活动。别忘了E-M5 Mark III 还有及其强劲的五轴防抖，如果不是要抓拍运动物体，这个高感已经完全够用了。我在瓜达拉加拉不太明亮的夜晚里（和国内的灯光比亮度差很多），ISO800 + F4已经能够完成绝大多数拍摄了。而此时，这台机器拍摄的画面还是非常干净的。来到GX9这边，实话实说还是让人有点失望的。相对于GX7，低感的进步是最大的，终于没有了100%怎么看都有的蠕虫斑，这点其实意义还是很大的。至于高感，和GX7相比进步非常微小，而和E-M5 MarkIII来比似乎有着1档左右差距。对于同样是2000万像素的传感器来说似乎有点不应该。而RAW的宽容度同样不能有太高的期望，即便是ISO200，提高2个EV也有一些噪点产生，虽然说并不明显。所以整体来说E-M5 Mark III在画质的高感和低感宽容度上，相对于GX9都是有优势的，如果你要寻求的是一台配合12-100f4这种旅游镜头出门采风的机器，无论从画质、色彩还是防抖能力来说，E-M5 Mark III都是更好的选择。

最后再谈谈M43的强项对焦吧。虽然不知道这台机器对焦速度具体的数值，但是你完全不需要去担心。无论是按快门，还是使用屏幕触摸释放快门，都是随点随有。而且几乎是在任何场合，哪怕是灯光极其微弱的地方（比如前面提到的小区游乐场）。这台机器在我手上使用至今两周多，还没有遇到过拉风箱或者对焦犹豫的时候。奥林巴斯这个几乎布满全屏幕的对焦点都是十字的，提高了对于某些图形的对焦能力，比我之前用过的其他M43相机都还要靠谱。而且但凡相机显示合焦的，最终回放查看焦点都是对实的。虽然现在市面上不少无反相机的对焦速度看指标也都不慢，但是具体使用中会在弱光下无法顺利对焦，甚至于显示合焦却会发现实际焦点不实。相比而言，这台 E-M5 Mark III 在对焦方面是完全让人放心的。GX9并不具备相位对焦能力，虽然说松下的反差对焦是独步天下的，但缺少了相位对焦的支持，GX9在逆光下或者极端弱光下的合焦能力是不如E-M5 Mark III的。当然即便如此还是比市面上的绝大多数相机对焦更灵敏，对于实际使用来说绝对是够用的。但是，GX9的机械快门引入了电磁快门，这个电磁快门会比GX7的纯机械快门有更大的延迟。所以虽然相机合焦了，但是总要等那么一小下才会真正释放快门。这点对于运动过程中的抓拍会带来不小的影响，以至于后来我在街头抓拍走动的人的时候都选择使用电子快门来减小快门时滞。回到电子快门以后，松下GX9又变回了我所熟悉的那台松下，随按随有，犀利无比。

这台相机还有不少特色功能，其中专业抓拍模式以及长曝光实时显示有点意思。这台机器的连拍速度本来就很不错，而这个专业抓拍模式快门的触发更灵敏（似乎是半按快门的时候就能触发），无形中提高了手脑响应速度，让用户能够更大几率地抓到自己想要的画面。不过这台相机的机内存储容量似乎不是很大，也许也是因为我用的SD卡速率也不是很高的原因，在连拍两秒后就会出现降速。此外长曝光实时显示是一个很实用的功能。在使用脚架进行长时间B门（或者T门）拍摄的时候，相机会每隔0.5s~60s（可以自行设定）来显示目前的曝光结果。用户可以随时依据曝光结果来决定是否要停止曝光。这对于时常要估算曝光时间并一次次试拍的风光摄影者来说带来了很大的便利，而且也避免了在最佳拍摄时机因为曝光问题错过了自己所需要的最佳图片的可能。（这些经历在我身上曾经多次发生，当然相信对于专业的风光摄影师来说应该不会犯这样的错误）松下包括iA模式在内的很多智能功能是我一直很喜欢的，尤其是它自动判断摄者意图并自动寻找画面中合适的对焦点的能力。很多时候根本不需要去选择对焦点，能做到框住画面就拍。当然在引入触摸屏和触摸快门之后这种优势渐渐地被弱化了，但依然是松下很招人喜欢的一个特点，至少我还可以很随性地拍摄，而相机总会给我一个还不错的结果。相对而言，用奥林巴斯E-M5 Mark III的时候要相对严谨一些，尽量都靠自己去控制那些拍摄细节。只要控制好了，它带来的结果会更好。

整体而言，这两台M43相机各有特色，如果更愿意慢一点控制好细节之后再拍摄，那么E-M5 Mark III会给你带来更好的结果。而GX9作为一个廉价的选择，也能在绝大多数时间给你一个不错的结果，而且也能让你更随性和快速的攫取瞬间的画面。当然，前提是不介意关掉它的电磁快门的情况下。用完GX9，更觉得E-M5 Mark III是一台好相机，在M43这样的底上，奥林巴斯已经做得很努力了。也许因为还有L口全幅，松下似乎在GX9上显得有那么些敷衍。

12-100f4真的是一支好镜头。配上一台小巧的M43机器是一个靠谱的旅行拍摄选择。无论我用E-M5 Mark III在瓜达拉哈拉还是用GX9在杭州，它都能胜任我所需要的画面拍摄。在GX9上使用这支镜头带来的防抖效果比机身防抖的效果要好。
```
